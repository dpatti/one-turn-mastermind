How to play Mastermind

Mastermind is a classic game where you try to guess a secret code made up of four colors. The order of the colors matters, and colors can repeat. If you're familiar with Wordle, it plays out in a similar style.

In normal play, a player would submit a guess and receive a number of black and white pegs in response to their guess. Each black peg tells you that one slot of the guess is correct, though it does not tell you which one. Similarly, each white peg tells you that one slot contains a correct color but with an incorrect position. A player continues to submit guesses until they get the secret code or run out of turns. Repeated colors in a guess may each be rewarded a black or white peg up to the number of times that color appears in the code. For example, if a guess is "BBRB" and the secret is "BYBP", one black peg would be given for the first "B", and one white peg would be given because another "B" was in the wrong location.

In this variation on the game, you are given a few rounds of guesses and can make one guess for the final secret code. There is always enough information in the initial guesses that the code can be determined logically. See the README on GitHub for some more information and hints.
